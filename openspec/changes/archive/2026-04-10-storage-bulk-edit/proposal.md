# Storage Bulk Edit — 全域套用面板

## 需求

在庫位管理列表頂部新增一個可收合的「全域套用」面板，讓使用者可以一次性更新所有庫位的：
1. 寬度（width）和深度（depth）
2. 第 1 層取料深度（pickDepth）和放料深度（placeDepth）
3. 第 2 層取料深度（pickDepth）和放料深度（placeDepth）

## 設計

- 面板預設**收合**，點標題展開
- 初始值**空白**（不顯示任何預設值）
- 空白欄位**不套用**（跳過該欄位）
- 兩個獨立的套用按鈕：「套用尺寸到全部」和「套用深度到全部」
- 套用深度時，只有 `layers === 2` 的庫位才更新第 2 層

## 影響檔案

- `src/components/StorageManager.tsx` — 加入全域套用面板 UI 和套用邏輯
